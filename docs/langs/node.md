# Node.js

Add a [`package.json`](https://docs.npmjs.com/files/package.json) to your bundle to specify versions for Node.js and/or required packages.

e.g.

```json
{
	"engines": {
		"node": "7.9.0"
	},
	"dependencies": {
		"to-camel-case": "1.0.0",
		"is-number": "3.0.0"
	}
}
```
